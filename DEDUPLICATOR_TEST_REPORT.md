## 去重算法验证

### Phase 1: 归一化 ✅
- 去除特殊字符：ARV-471 → arv471
- 去除空格：BGB 3245 → bgb3245
- 转小写：ARV471 → arv471
- 中文映射：阿维替尼 → arv471（通过上下文推断）

### Phase 2: 聚类 ✅
- 主键：(normalized_drug_name, normalized_entity_name)
- Cluster 1: (arv471, arvinas) - 3 条记录
- Cluster 2: (bgb3245, beigene) - 2 条记录

### Phase 3: 合并 ✅
- 标准名称：选择英文 INN 或最常见名称
- 别名：收集所有非标准名称
- 国家：union 操作（US, CN, JP）
- 临床阶段：取最先进阶段
- 证据引用：选择最权威来源（专利 > 新闻）

### Phase 4: 排序 ✅
- 按临床阶段降序：Phase I > IND-Enabling > Pre-Clinical
- 按降解剂类型优先级：LYTAC > PROTAC > Molecular Glue
